Mis Proyectos

01

Web

Junior Shop

Tienda de Mascotas

Este proyecto consiste en la creación de un sitio web completo para una tienda de mascotas llamada Junior Shop. El objetivo es transformar un prototipo en papel en una página web funcional, utilizando las mejores prácticas de desarrollo web con HTML y CSS. El resultado final es un sitio web atractivo. Los usuarios pueden navegar fácilmente por las categorías de productos, el sitio es rápido, moderno y optimizado para SEO, con una experiencia fluida en cualquier dispositivo.

02

Web

Memory Game

Juego de Memoria

Este proyecto es un juego de memoria interactivo creado utilizando JavaScript, HTML y CSS, como parte de un curso sobre los fundamentos del lenguaje de programación JavaScript. El juego está basado en las imágenes de los personajes de la famosa serie de televisión Los Simpson y consume una API de frases de la serie para generar un contenido dinámico en el juego. Se implementó jQuery para simplificar la manipulación del DOM.

03

Web

App

App de Productividad

Sistema de gestión de tareas con MySQL, el sistema permite a los usuarios registrar, consultar, actualizar y eliminar tareas asociadas a su cuenta. Cada tarea incluye información como el título, descripción, estado, y hora de creación, con relaciones entre usuarios y tareas para asegurar integridad. Las consultas SQL permiten filtrar tareas según el usuario, visualizar tareas pendientes o completadas, y gestionar información de forma eficiente.
